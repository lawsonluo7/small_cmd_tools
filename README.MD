# Some Small Command line tools
## Commands
### calc
**usage:** `python calc.py [-h] equation`

Evaluate a mathematical equation.

#### Operators:
`+`, `-`, `*`, `/`, `**`, `%`, `//`, `=`

#### Arguments (in terminal)
##### positional arguments:
  `equation`        The equation to evaluate

##### options:
  `-h`, `--help`      show help message and exit\
